# Web Traffic Conversion Prediction

## 📌 Project Overview
This project uses machine learning to predict user conversion behavior based on web traffic data.

## 📊 Dataset Features
- session_duration_sec
- pages_viewed
- bounce
- device_type
- browser
- traffic_source

## ⚙️ Steps Followed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Feature Engineering
- Handling Class Imbalance (SMOTE)
- Model Training (Logistic Regression, Random Forest, XGBoost)
- Model Evaluation (Confusion Matrix, ROC-AUC)

## 🤖 Models Used
- Logistic Regression
- Random Forest
- XGBoost

## 📈 Results Summary
- Accuracy improved after feature engineering
- Recall for minority class improved
- AUC shows moderate predictive power

## 🚀 Key Insight
User behavior is complex and requires non-linear models to capture patterns effectively.
